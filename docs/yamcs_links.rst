yamcs links
===========

.. program:: yamcs links

Synopsis
--------

**yamcs links list**

**yamcs links enable** <*LINK*>...

**yamcs links disable** <*LINK*>...

**yamcs links describe** <*LINK*>


Description
-----------

Read and manipulate data links.


Commands
--------

list
    List links

enable
    Enable a link

disable
    Disable a link

describe
    Describe a link

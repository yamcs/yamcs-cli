yamcs storage cp
================

.. program:: yamcs storage cp

**NAME**

    yamcs storage cp - Copy a file or object


**SYNOPSIS**

    ``yamcs storage cp SRC DST``


**POSITIONAL ARGUMENTS**

    .. option:: SRC

        local file or an object in the format BUCKET://OBJECT.

    .. option:: DST

        local file or an object in the format BUCKET://OBJECT.

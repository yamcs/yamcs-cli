Yamcs Command-Line Interface
============================

Getting Started
---------------

The Yamcs Command-Line Interface is written in Python and distributed separately from Yamcs Server.

Install with pip::

    pip install --upgrade yamcs-cli


Run :doc:`yamcs_login` to initialize your environment:

.. code:: shell

    yamcs login

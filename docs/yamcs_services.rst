yamcs services
==============

.. program:: yamcs services

Synopsis
--------

.. rst-class:: synopsis

    | **yamcs services** list
    | **yamcs services** start <*SERVICE*>...
    | **yamcs services** stop <*SERVICE*>...


Description
-----------

Read and manipulate services.


Commands
--------

.. describe:: list

    List services

.. describe:: start <SERVICE>...

    Start a service

.. describe:: stop <SERVICE>...

    Stop a service

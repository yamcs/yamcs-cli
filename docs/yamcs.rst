yamcs
=====

.. program:: yamcs

Synopsis
--------

.. rst-class:: synopsis

    | **yamcs** [--version] [-h, --help] <*COMMAND*> [<*ARGS*>]


Description
-----------

Command-line interface for Yamcs.

Run ``yamcs login`` to initialize your environment.


Options
-------

.. option:: --version

    Prints the CLI version and quit.

.. option:: -h, --help

    Prints the synopsis and the list of commands.
